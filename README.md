# CO-OP
AngularJS chat team like SLACK, using REST API. Desktop app make with electron.

#Features

- Join the team
- Login to the team
- Delete members
- Create channels
- Delete channels
- Update channels
- Join channels
- Post message
- Update message
- Delete message
